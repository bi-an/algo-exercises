1. net-tools

   > 网络诊断工具。


2. alacarte

   > 收藏夹Dock管理器（主菜单）。使用这个 GUI 工具，免去了每次都手动添加 ` .desktop` 文件的麻烦。

   ```bash
   sudo apt install alacarte
   ```
