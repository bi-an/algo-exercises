## 时间复杂度

- `O(n)`的算法数据范围`n < 10^8`。
- `O(n * logn)`的算法数据范围`n <= 10^6`。
- `O(n * sqrt(n) )`的算法数据范围`n < 10^5`。
- `O(n^2)`的算法数据范围`n < 5000`。
- `O(n^3)`的算法数据范围`n < 300`。
- `O(2^n)`的算法数据范围`n < 25`。
- `O(n!)`的算法数据范围`n < 11`。
